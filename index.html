Bienvenue sur Digital Student ! Vous trouverez sur ce site diverses informations sur HTML5 et CSS3. Que ce soit l'intérêt d'apprendre HTML/CSS, les balises de base mais aussi des tips et bonnes pratiques !

Pourquoi apprendre HTML/CSS

HTML et CSS sont deux langages relativement simple à comprendre et à utiliser. Ils forment une bonne porte d'entrée vers les langages de programmation. Leur connaissance est indispensable étant donné qu'ils composent la base de tous site web. De plus, la maitrise de HTML et CSS permet de créer des sites web personnalisés et mais surtout, accessible pour tous !

Les balises

<h1>

La balise h1 indique un titre sur un site web. Elle joue un rôle dans le référencement du site. Elle peut être déclinée de 1 à 6 (h1, h2, h3, etc).

Exemple :

<h1>Titre H1</h1>

Titre H1

<p>

La balise p permet de définir un paragraphe.

Exemple :

<p>ceci est un texte normal</p>

ceci est un texte normal

<a>

La balise a est un élément d'ancrage. Son attribut HREF permet de créer un lien hypertexte vers des pages web, des fichiers, etc.

Exemple :

<a href="#intro">Introduction</a>
Introduction

<img>

La balise img permet d'intégrer une image.

Exemple :

<img src="assets/images/logoicon.svg">

<ul>

La balise ul permet de créer une liste sans ordre particulier contrairement à la balise ol. ul est souvent représenter avec des puces.

Exemple :

<ul><li>Liste à puces</li></ul>
  • Liste à puces

<li>

La balise li permet de représenter un élément dans une liste. Elle est donc contenu dans une balise ul ou ol.

Exemple :

<ul><li>Élément 1</li><li>Élément 2</li><li>Élément 3</li></ul>
  • Liste à puces
    • Élement 1
    • Élement 2
    • Élement 3

Tips et bonnes pratiques

HTML

    Référencement

    • Mets des mots clés dans tes titres
    • Les sections servent au référencement contrairement aux div

    Accessibilité

    • Utilises un bouton plutôt qu'une div dans un formulaire pour permettre une navigation complète au clavier
    • Utilises l'attribut alt pour décrire tes images, cela permettra de référencer l'image mais aussi de faciliter les personnes mal-voyantes
    • Choisis bien le format de tes média pour optimiser les performances de ta page

CSS

    Organisation

    • Commences ton code par un reset CSS
    • Organises ton CSS de manière cohérente
    • Utilises les sauts de ligne pour aérer ton code
    • Commentes ton code
    • Sépare ta stylesheet en plusieurs partie si tu travailles sur un gros projet
